# Design System (Cart Lift)

## Overview

Cart Lift follows WordPress admin design patterns with email template design for customer-facing recovery campaigns.

## UI Components

### WordPress Admin

- Standard WordPress admin components
- Analytics dashboards
- Campaign management interface
- Cart list tables

### Email Templates

- Responsive email design
- Product display cards
- Dynamic discount codes
- Call-to-action buttons

## Color Palette

### Admin Interface

- Follows WordPress admin color scheme

### Email Templates

- **Primary CTA:** #0073aa (or customizable)
- **Success:** #46b450
- **Urgency:** #dc3232
- **Background:** #f7f7f7

## Email Design Principles

### Mobile-First

- Responsive email layouts
- Touch-friendly buttons (minimum 44px)
- Readable font sizes (minimum 14px)

### Brand Customization

- Logo upload
- Color scheme customization
- Custom footer text
- Social media links

### Best Practices

- Plain text alternatives
- Inline CSS for compatibility
- Tested across email clients
- Spam filter optimization

## Cart Display

### Product Cards

- Product image
- Product name
- Quantity
- Price
- Brief description

### Recovery CTA

- Clear "Complete Purchase" button
- Discount code display (if applicable)
- Urgency messaging
- Trust indicators

## Analytics Dashboard

### Key Metrics

- Total abandoned carts
- Recovery rate
- Revenue recovered
- Email campaign performance

### Charts & Graphs

- Line graphs for trends
- Bar charts for comparisons
- Donut charts for percentages
- Data tables for details

## Accessibility

- WCAG AA compliance for admin
- High contrast email templates
- Alt text for images
- Semantic HTML

See the [Best WooCommerce Feed Design System](../../best-woocommerce-feed/docs/design/design-system.md) for detailed WordPress admin design guidelines.
